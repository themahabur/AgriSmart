# AgriSmart - Smart Farming Solutions Platform

## About AgriSmart

AgriSmart is a comprehensive agricultural platform designed to empower farmers with modern technology and data-driven insights. The platform serves as a one-stop solution for farmers to access real-time market prices, weather forecasts, expert consultations, AI-powered farming advice, and a community forum to share experiences.

## Key Features

- **AI-Powered Assistance**: Get farming advice in Bangla through our smart chatbot
- **Market Insights**: Real-time commodity prices and trend analysis
- **Weather Forecasting**: Agricultural advisories based on location
- **Expert Consultations**: Connect directly with agricultural experts
- **Farm Management**: Track crops, tasks, and yields
- **Knowledge Hub**: Access to blogs and educational content
- **Community Forum**: Share experiences with fellow farmers
- **Government Schemes**: Information on subsidies and assistance programs

## Technology Stack

- **Framework**: Next.js 15.5.3 with App Router
- **Frontend**: React 19.1.0, Tailwind CSS
- **Authentication**: NextAuth.js
- **AI Integration**: OpenRouter API
- **Image Management**: Cloudinary
- **UI Components**: React Icons, Recharts, Swiper
- **Rich Text Editor**: Tiptap

## Project Structure

The application is organized into two main sections:

1. **Public Site** - Informational pages for unauthenticated users
2. **Dashboard** - Feature-rich platform for authenticated farmers

For detailed information about the project structure and all features, please refer to [PROJECT_DOCUMENTATION.md](PROJECT_DOCUMENTATION.md).
